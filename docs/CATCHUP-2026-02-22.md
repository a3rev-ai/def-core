## To align your local

```bash
cd def-core
git checkout main
git pull
npm install
```

To run the test suite (requires Docker):

```bash
php tests/run.php              # Unit tests (no Docker)
npm run env:start              # Start wp-env containers
npm run smoke                  # Smoke test
npm run test:phpunit           # PHPUnit integration tests
npm run env:stop               # Stop containers
```

See `docs/TESTING.md` for the full command reference.
